# 点金术选股系统手机版使用教程

## 1. 打开网址

手机浏览器打开：

```text
https://diwuhai77-maker.github.io/dianjinshu-mobile/
```

这是公网地址。外出时可以直接打开，不需要电脑在身边，也不需要手机和电脑连接同一个 Wi-Fi。

## 2. 添加到小米手机桌面

1. 用小米浏览器或 Chrome 打开上面的网址。
2. 点浏览器右上角菜单。
3. 选择“添加到桌面”或“安装应用”。
4. 以后从桌面点“点金术”即可打开。

## 3. 页面怎么看

- “总览”：看今天 ETF 买点、A 股严格/备用数量、检查了多少只股票。
- “ETF”：看 562500、512760、563230、159326 是否到点金术买点。
- “A股”：分开看“严格候选”和“备用观察候选”。

## 4. A 股两个区的区别

“严格候选”使用完整点金术条件：

- PE 小于 20
- 股息率大于 3%
- ROE 大于 10%
- 市值大于 50 亿
- 利润增长为正
- 当前价低于 MA120 的 88%
- 非 ST

“备用观察候选”只在实时源缺少 PE、市值、股息率等字段时出现。它主要参考：

- ROE 大于 10%
- 利润增长为正
- MA120 价格位置
- 非 ST

备用观察候选只是外出时先看的观察名单，不等同于严格候选。真正按完整条件执行的结果只看“严格候选”区。

## 5. 总览数字

A 股数量会显示成：

```text
严格候选数量 / 备用观察数量
```

例如 `0/46` 表示今天没有完整严格候选，但有 46 只备用观察候选。

## 6. 更新时间

系统设置为交易日自动更新：

```text
北京时间 17:30 左右
```

页面顶部时间就是最近一次生成数据的时间。

## 7. 手机刷新

如果手机看起来还是旧内容：

1. 下拉刷新页面。
2. 或在网址后面加 `?v=3` 再打开：

```text
https://diwuhai77-maker.github.io/dianjinshu-mobile/?v=3
```

## 8. 注意

这个工具只做固定规则筛选和提醒，不构成投资建议。买卖前仍要结合仓位、风险承受能力和市场环境判断。
