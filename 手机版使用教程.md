# 点金术选股系统手机版使用教程

## 1. 打开网址

手机浏览器打开：

```text
https://diwuhai77-maker.github.io/dianjinshu-mobile/
```

这是公网地址。外出时可以直接打开，不需要电脑在身边，也不需要手机和电脑连接同一个 Wi-Fi。

## 2. 添加到小米手机桌面

1. 用小米浏览器或 Chrome 打开上面的网址。
2. 点浏览器右上角菜单。
3. 选择“添加到桌面”或“安装应用”。
4. 以后从桌面点“点金术”即可打开。

## 3. 页面怎么看

- “总览”：看今天有多少 ETF 买点、多少 A 股候选、检查了多少只股票。
- “ETF”：看 562500、512760、563230、159326 是否到点金术买点。
- “A股”：看今天筛出来的买点候选或观察候选。

## 4. A 股候选说明

严格数据源正常时，会优先使用这些条件：

- PE 小于 20
- 股息率大于 3%
- ROE 大于 10%
- 市值大于 50 亿
- 利润增长为正
- 当前价低于 MA120 的 88%
- 非 ST

如果页面显示“观察候选”，说明当前没有严格买点，或者实时源缺少 PE、市值、股息率等字段。系统会退一步，先用 ROE、利润增长、MA120 找出最接近买点的股票，方便你外出时先看名单。

## 5. 更新时间

系统设置为交易日自动更新：

```text
北京时间 17:30 左右
```

页面顶部时间就是最近一次生成数据的时间。

## 6. 手机刷新

如果手机看起来还是旧内容：

1. 下拉刷新页面。
2. 或在网址后面加 `?v=1` 再打开：

```text
https://diwuhai77-maker.github.io/dianjinshu-mobile/?v=1
```

## 7. 注意

这个工具只做固定规则筛选和提醒，不构成投资建议。买卖前仍要结合仓位、风险承受能力和市场环境判断。
